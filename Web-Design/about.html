Overview

The primary purpose of this website is to provide a foundation to create webpages with the functionality available in HTML 5 and CSS3.

This website's missions is to provide the informaiton required to create a improved internet and allow the next generation of developers and content creators to create web content.

The amount of syntax available for HTML 5 and CSS 3 can be difficult to understand or remember, this website puts all the most used/crucial syntax on a single page, so if you ever want to know how something works, just go to the blog page.
